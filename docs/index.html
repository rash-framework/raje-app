Discover RAJE

RAJE is a stand-alone, multi-platform software used for writing HTML articles natively.
It comes with a user-friendly interface that hides all the complexities of writing HTML lines from scratch.

It has been designed for:

Being totally compatible with HTML5, RASH and the existing tools developed for the Framework

Semplifiying the writing of an article keeping everything as natural as possible

Not enabling the user to personalise the rendering of the document – focus on the content only

Following best practices used in existing word processors for creating structures typical of papers

Allowing the creation of mathematical formulas with an appropriate interface, supporting Web-first formats

Interacting with GitHub for easing the storing, sharing, and preservation of the articles
